Collections

Collections Service is used to send the e-collection transaction(NEFT/RTGS/IMPS) details to client and receive response from client for every transaction.

Services Description
iSure Pay iSurePay API is used to enable clients to authenticate the cash, cheque, and DD depositor in branch and update the deposited amount to the client's server.
E Collection This API is used to send the e-collection transaction details to client and receive response from client for every transaction.
