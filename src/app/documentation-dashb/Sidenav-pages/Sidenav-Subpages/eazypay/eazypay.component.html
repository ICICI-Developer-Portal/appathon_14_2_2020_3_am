EazyPay

'Eazypay is a first of its kind secure payment service by ICICI Bank in India. It enables institutions to collect money from their customers through multiple payment modes. The APIs allow merchants to have a complete transactional flow ranging from QR code generation for the merchant, send collect request, check transaction status, and claim for refund.'

API Description
QR Code This API is used to generate a reference ID which is required in QR code generation
Collect Pay This API is used to generate the collect pay request for payee. Collect pay approval request will be sent to PSP app
Transaction Status This API is used for checking the status of the transaction.
Refund This API is used for refund of the transaction.
Callback Status This API is used to check the callback status.
