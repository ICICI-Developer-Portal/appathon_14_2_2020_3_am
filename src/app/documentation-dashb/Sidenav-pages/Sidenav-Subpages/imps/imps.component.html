IMPS

IMPS Service enables client to make payment through IMPS mode. Client can also get the transaction status through API.

Services Description
P2P - Person to Person Fund Transfer Enables funds transfer from remitter BC account to beneficiary account using MMID and MobileNumber.
P2A - Person to Account Fund Transfer Enables funds transfer from remitter BC account to beneficiary account using IFSC and Account Number details of Beneficiary.
IMPS MMID Inquiry This is inquiry transaction to know the status for MMID generation.
IMPS Transaction Inquiry This service will return the status of transaction specific to given Reference Retrieval Number.
IMPS Transaction History This service will return transaction history details of last six month from todayâ€™s date for financial transactions.
IMPS Transaction Status This service is used for checking the status of the transaction.
Fund Transfer To transfer funds from one ICICI account to another ICICI account. Each request allows one transaction only.
